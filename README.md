### 🤖 [Multi-Agent Faculty Support System](https://github.com/orwajabali/multi-agent-rag-chatbot) · 2025
> *M.Sc. Thesis Project — Production-Grade Agentic AI*

A pedagogically context-aware multi-agent system designed for enhanced faculty support in Higher Education. Implements a full agent pipeline: **Guard → Classification → Content Generation → Assessment Generation → Submission Assessment**, with RAG retrieval and LangGraph orchestration.

**Stack:** `Python` `LangChain/LangGraph` `Pinecone` `LLM APIs` `HuggingFace` `RunPod` `Docker` `MCP`

### 🏫 [Digital Twin Presentation Trainer](https://github.com/orwajabali/Smart_Systems_Management_Chatbot) · 2025
> *AI-Assisted Feedback System for Presenters*

An intelligent feedback system that analyzes presenter timing, engagement cues, and performance indicators in real time — bridging digital twin simulation with educational AI.

**Stack:** `Python` `Signal Processing` `Real-time Dashboards`

---

### 🌊 [SafeRoute Stream](https://github.com/orwajabali/saferoute-stream) · 2024
> *Real-Time Streaming ML Pipeline*

End-to-end real-time road accident severity prediction pipeline. Architecture: **Kafka → Spark Structured Streaming → MongoDB → Live Leaflet Heatmap Dashboard** with sub-second prediction latency.

**Stack:** `Apache Kafka` `Apache Spark` `MongoDB` `Python` `Scala`

---

### 📰 [Scientific Paper Title Recovery](https://github.com/orwajabali/scientific-paper-title-recovery) · 2024
> *NLP / Fine-Tuning*

NLP pipeline for recovering missing titles from ~1,000 noisy scientific papers. Fine-tuned **T5** transformer for abstractive title generation, with Arabic NLP (NER & Text Classification) using MaxEnt on ANERcorp dataset.

**Stack:** `PyTorch` `HuggingFace Transformers` `T5` `scikit-learn` `ANERcorp`

---

### 🐱 [Cat Breed Classification](https://github.com/orwajabali/cat-breed-classification) · 2023
> *Computer Vision Benchmark*

Benchmarked two CV approaches: **Custom CNN (75.98%)** vs. **YOLOv8 segmentation + LightGBM feature engineering (81.81%)**. Demonstrated how combining object segmentation with classical ML improves classification accuracy.

**Stack:** `PyTorch` `TensorFlow` `Keras` `YOLOv8` `LightGBM` `OpenCV`

---

### 🛣️ [UK Road Safety Mining](https://github.com/orwajabali/uk-road-safety-mining) · 2023
> *Data Mining at Scale — 8.6M Records*

End-to-end data mining and EDA on 8.6 million UK road accident records. Applied feature engineering, clustering, and association rule mining to extract actionable safety insights.

**Stack:** `Python` `Pandas` `scikit-learn` `Matplotlib`

---

### 🏠 [House Price Prediction & Time Series](https://github.com/orwajabali/house-price-prediction) · 2023
> *ML from Scratch*

Built **Linear, Polynomial, Ridge & Lasso regression from scratch** using NumPy. Extended to time series forecasting on Melbourne temperature data comparing baseline Prophet vs. feature-engineered variants.

**Stack:** `Python` `NumPy` `Pandas` `Prophet`
